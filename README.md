This repository contains Python programs submitted as part of the Tutedude Python Assignment.

Each task demonstrates basic Python concepts through simple, beginner-friendly programs.

Task 1: Basic Mathematical Operations
Objective:
Create a Python program that:

Takes two numbers as input

Performs:
Addition
Subtraction
Multiplication
Division

Displays all results clearly

Task 2: Personalized Greeting
Objective:
Write a Python program that:

Takes a first name and last name as input

Concatenates them to form a full name

Prints a friendly personalized greeting

Files Included
File Name	Description
task1.py	Performs basic mathematical operations on two numbers
task2.py	Creates and prints a personalized greeting using full name
